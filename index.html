2. Data Visualization: Tide Table

Visual Resource

Screen Explanation: The "Monthly Table" tab renders a line chart (via Chart.js) that displays the maximum and minimum tide heights for each day, allowing for easy identification of patterns and tide peaks.

Relevant Code Snippet

Tide Data Processing (app.py)

The /tabua_mes route reads the local JSON file, processes the data, and structures it for the frontend to generate the chart.

# Route to get tide data for the monthly table
@app.route('/tabua_mes')
def get_tabua_mes():
    cidade = request.args.get('cidade')
    # (...)
    cidade_encontrada = next((item for item in dados_mare_json if item["cidade"].lower() == cidade.lower()), None)
    # (...)
    # Process data to extract daily high and low tides
    dados_grafico = {
        "labels": [], "altas": [], "baixas": []
    }
    for dia_mare in cidade_encontrada['mares']:
        dados_grafico["labels"].append(dia_mare['data'])
        
        altas = [ev['altura_m'] for ev in dia_mare['eventos'] if ev['tipo'] == 'alta']
        baixas = [ev['altura_m'] for ev in dia_mare['eventos'] if ev['tipo'] == 'baixa']
        
        dados_grafico["altas"].append(max(altas) if altas else None)
        dados_grafico["baixas"].append(min(baixas) if baixas else None)

    return jsonify(dados_grafico)

Unit Test Example (test_app.py)

This test, using pytest, simulates a call to the /clima route and validates that the server's response is correct, ensuring API stability.

# Test for the /clima route with a successful outcome (using a mock)
def test_get_clima_success(client, mocker):
    # Mock the external API response to avoid internet dependency
    mock_response = mocker.Mock()
    mock_response.status_code = 200
    mock_response.json.return_value = {
        "name": "Recife",
        "main": {"temp": 27.0},
        "weather": [{"description": "clear sky"}]
    }
    mocker.patch('requests.get', return_value=mock_response)
    
    # Execute the test request
    response = client.get('/clima?cidade=Recife')
    
    # Check the assertions (if the result is correct)
    assert response.status_code == 200
    data = response.get_json()
    assert data['cidade'] == 'Recife'
    assert 'temperatura' in data
